微信公众平台快速开发框架 For DotNet

WXPP Quick Framework For DotNet

一个使您能够快速开发微信公众平台的框架

快速开发框架已经升级到[V5]版本，目前已支持.Net Core 2.0，如果您是.Net Framework版本，您可以使用[V4]或以下版本!

项目地址

.Net Framework版本项目地址：https://github.com/JamesYing/JCWX
.Net Core 版本项目地址：https://github.com/JamesYing/JCWXCore

目前进度

2017-11-01:目前基本完善了API接口，已经支持被动消息加解密，但是Core版本的文件上传还未完成，稍后完成后会及时更新

必要条件

首先我们必须注册成为微信公众平台用户，目前账号分为订阅号和服务号，您可以点击这里查看如何注册成为平台用户。在非认证情况下所使用的是低级接口，只有在认证后的服务号才能享受高级接口服务。

如何使用

.Net Framework如何使用

  • 1、通过 git clone https://github.com/JamesYing/JCWX.git 命令获取完整项目进行编译
  • 2、通过 NuGet 获取，可搜索WXQucikFramework或使用控制台
    Install-Package JCSoft.WX.Framework -Version 4.0.0

.Net Core如何使用

  • 1、通过 git clone https://github.com/JamesYing/JCWXCore.git 命令获取完整项目进行编译
  • 2、通过 NuGet 获取，可搜索WXQucikFramework或使用控制台
    Install-Package JCSoft.WX.Framework -Version 5.1.1
